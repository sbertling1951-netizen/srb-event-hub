# FCOC Event Hub Admin Quick Start

## 1. Create the event
1. Open the app and go to **Admin > New Event**.
2. Enter the event name, dates, event code, and rally location.
3. Enter the registration close date and self-edit close date.
4. Save the draft.

## 2. Set the rally location
1. In the event record, enter the full rally address.
2. Save the location.
3. Use this location as the anchor for directions links and nearby places.

## 3. Import registrations
1. Go to **Admin > Imports**.
2. Upload a CSV or Excel file exported from your website or registration source.
3. Review the row count and preview table.
4. Check warnings for missing or duplicate values.
5. Import the file.

## 4. Assign RV sites
1. Open the attendee locator / parking view.
2. Review which sites are open and which are assigned.
3. Match attendees to sites before arrival day.
4. Update changes as late registrations come in.

## 5. Load agenda items
1. Add each session, meal, seminar, and activity time.
2. Enter the location name and any updated notes.
3. Mark a session as updated if the time or place changes.

## 6. Add optional activities
1. Create each tour, dinner, or excursion.
2. Set price, capacity, and signup cutoff date.
3. Add the departure or meeting location address.
4. Add an RV note where needed, such as “Use toad only” or “Shuttle pickup”.

## 7. Review nearby places
1. Add fuel, groceries, pharmacy, urgent care, and attractions.
2. Confirm addresses and phone numbers.
3. Make sure direction links work.
4. Add RV-friendly notes when helpful.

## 8. Post announcements
1. Add the message title.
2. Mark it urgent if the change affects timing or safety.
3. Publish it.

## 9. Final pre-event check
1. Confirm registration cutoffs are correct.
2. Confirm activity cutoffs are correct.
3. Confirm attendee list and parking assignments.
4. Confirm nearby links and phone numbers.
5. Confirm welcome message and first-day agenda are published.
