# FCOC Event Hub Admin Quick Start

## 1. Create the event
1. Open **Admin**.
2. Select **Events**.
3. Enter the event name, venue name, address, city/state, and event code.
4. Enter the registration close date.
5. Enter the self-edit, cancellation, refund, and planning lock dates.
6. Click **Save Draft**.

## 2. Add the location
1. In **Events**, verify the event address.
2. Save the address exactly as you want members to use it for directions.
3. Later, this address can drive the Nearby list and directions buttons.

## 3. Import registrations
1. Open **Admin > Imports**.
2. Upload the CSV or XLSX file exported from your website registration system.
3. Review the row count and warnings.
4. Confirm names, emails, and site numbers look correct.
5. In a production build, the next step would save the records to the event roster.

## 4. Add optional activities
1. Open **Admin > Activities**.
2. Enter the activity name and description.
3. Enter the meeting location.
4. Set the capacity and price.
5. Set the signup, waitlist, cancellation, and refund deadlines.
6. Save the activity.

## 5. Add announcements
1. Open **Admin > Announcements**.
2. Enter a title and message.
3. Set the priority.
4. Set the expiration time if needed.
5. Publish the message.

## 6. Review nearby places
1. Open the member **Nearby** screen.
2. Test Apple Maps and Google Maps links.
3. Confirm that urgent care, pharmacy, fuel, and groceries are useful for attendees.
4. Add RV-friendly notes where needed.

## 7. Final pre-event check
1. Review the Home screen.
2. Review the Agenda screen.
3. Review the Activities screen for open/closed status.
4. Review the Attendee Locator.
5. Confirm all cutoff dates are correct.
6. Confirm the event code is correct for attendees.

## Recommended operating rule
Keep attendee registration and optional activity registration separate. That will make counts, refunds, waitlists, and planning much easier to manage.
